coreutils/id - لتحديد هوية المستخدم

Youssef Hesham 2022-06-30 1 min read unlisted Permalink
  • #tldr
  • #linux
  • #coreutils
$ id

اداة id تستخدم للاستعلام عن ال ID وال group المشترك به المستخدم الحالي.

طريقة الاستعمال

لو كان اسم المستخدم الحالي youssef.

$ id
uid=1000(youssef) gid=1000(youssef) groups=1000(youssef),10(wheel),100(users)

الرقم 1000 هو رقم الuser ويبداء لينكس العد من رقم 1000 حتى 1999. لكن ده للمستخدمين العاديين, لكن عند مستخدم الroot بيكون رقمه دايما صفر (0)

$ echo $USER
root

$ id
uid=0(root) gid=0(root) groups=0(root) ...

لطباعة ID الgroups فقط.

$ id
uid=1000(root) gid=1000(root) ... 10(wheel), 18(audio)

$ id -G
1000 ... 10 18
